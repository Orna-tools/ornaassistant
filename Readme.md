# 🏰 Orna Assistant

**Orna Assistant** is an Android companion app for the mobile game **Orna: The GPS RPG**, providing overlays, notifications, and additional features to enhance gameplay.

![Orna Assistant Logo]

---

## ✨ Features
- 📊 **Overlays**: Dungeon stats, wayvessel tracking, invites, and more!
- 🔔 **Notifications**: Track important cooldowns and events.
- 🔄 **Automatic Updates**: Check for new versions and install them directly.
- 🎨 **Custom Themes**: Switch between Light, Dark, and System themes.
- 🛠️ **Discord Integration**: Generate kingdom gauntlet messages for Discord.
- 🔍 **Item Assess Overlay**: Automatically assess items in the inventory.
- 📅 **Session Tracking**: Keep track of earned orns and gold from dungeons.
